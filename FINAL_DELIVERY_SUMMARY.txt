1️⃣  Visiteur → http://localhost:3000/
    └─ Voit Hero + dernières vidéos
    └─ Clique "Créer ma vidéo"

2️⃣  Redirigé → /generate
    └─ Saisit prompt
    └─ Clique "Générer"

3️⃣  API vérifie cache
    ├─ HIT: Redirection immédiate vers /v/[id]
    └─ MISS: Job créé, redirection vers /v/[id]

4️⃣  Page /v/[id]
    ├─ Si cached: Vidéo immédiate
    └─ Si nouveau: Polling 8s + barre progression

5️⃣  Worker traite (4-9 min)
    ├─ Qwen → Script
    ├─ WAN Image → Visuel
    ├─ WAN Video → 4 clips
    ├─ ElevenLabs → Audio + SRT
    ├─ Remotion → Assemblage
    └─ Status: done

6️⃣  Polling détecte done
    └─ Affiche vidéo avec player

7️⃣  Utilisateur
    ├─ Regarde vidéo
    ├─ Télécharge
    ├─ Copie lien
    └─ Crée nouvelle vidéo
