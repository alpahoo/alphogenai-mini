# AlphoGenAI Mini - Status

## ✅ ARCHITECTURE MIGRATION COMPLETE

### Stack
- **Video Generation**: Runway Gen-4 Turbo
- **Script Generation**: Qwen Mock (no API calls)
- **Music**: Free tracks from Supabase Storage (YouTube Audio Library)
- **Worker**: Simplified async orchestrator (no LangGraph)

### Workflow
1. User submits prompt → Job created in Supabase
2. Worker polls jobs table
3. Orchestrator executes:
   - Qwen Mock generates script structure
   - Runway Gen-4 generates video (10s, 16:9)
   - Music selected from Supabase Storage by tone
   - Job marked as completed with final video URL
4. User views video at /v/{jobId}

### Files Structure
- `workers/runway_orchestrator.py` - Main workflow orchestrator
- `workers/runway_service.py` - Runway Gen-4 API wrapper
- `workers/qwen_mock_service.py` - Mock script generator
- `workers/music_selector.py` - Music selection from Storage
- `workers/worker.py` - Background job processor
- `workers/supabase_client.py` - Database operations
- `workers/config.py` - Configuration management

### Environment Variables
Required:
- `RUNWAY_API_KEY` - Runway Gen-4 Turbo API key
- `SUPABASE_URL` - Supabase project URL
- `SUPABASE_SERVICE_ROLE_KEY` - Supabase service role key

Optional:
- `QWEN_MOCK_ENABLED=true` - Use mock mode (default)
- `WEBHOOK_URL` - Webhook for job completion notifications

### Testing
Run: `python -m workers.runway_orchestrator "Test prompt"`
