# AlphoGenAI Mini — STATUS (V1 — source de vérité)

## Pipeline (happy path only)
- **UI**: `/generate` → crée un job (POST `/api/jobs`)
- **Worker**: poll `jobs.status=pending` → génère la vidéo → update `jobs.output_url_final` → `done`
- **Video backend (unique)**:
  - **Prod**: `ModalBackend` via `MODAL_VIDEO_ENDPOINT_URL`
  - **Local/CI**: `MockBackend` (`VIDEO_BACKEND=mock`) → MP4 dummy ~1s + upload Supabase
- **Storage**: Supabase Storage bucket **`generated`** en **public read** (V1, pas de signed URL)
- **Cache**: `video_cache.prompt_hash = sha256(JSON stable prompt+duration_sec+fps+resolution+seed)`

## Off / Legacy
- **Audio**: OFF (archivé dans `legacy/`)
- **Runpod / SVI / Runway / Replicate / Remotion / musique**: LEGACY uniquement (dans `legacy/`)

## Routes actives (V1)
- Pages: `/`, `/generate`, `/jobs/[id]`
- Redirections legacy: `/creator/generate` → `/generate`, `/v/[id]` → `/jobs/[id]`, `/dashboard` → `/generate`
- API: `POST /api/jobs`, `GET /api/jobs/[id]`, compat `POST/GET /api/generate-video` (wrapper strict)
